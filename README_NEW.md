## 🚀 Development

### Prerequisites
- Node.js 18+ 
- npm atau yarn

### Installation
```bash
# Clone repository
git clone https://github.com/caklem/Holy-fun.git
cd Holy-fun

# Install dependencies
npm install

# Run development server
npm run dev
```

### Build for Production
```bash
# Build website
npm run build

# Preview build
npm run preview
```
